QUIZ ENGINE DASHBOARD - SETUP
==============================

1. RUN THIS SQL:
   Open setup.sql and run it in your defaultdb database

2. LOGIN:
   Login to your account on the website

3. GENERATE QUIZ:
   Upload a text file and generate a quiz

4. CHECK DASHBOARD:
   Go to dashboard.html to see your saved quizzes

THAT'S IT!

The quiz will auto-save when you:
- Generate it (saves as "in_progress")
- Complete it (updates score to "completed")

Check browser console (F12) for any errors.
