QUIZ SAVE INTEGRATION GUIDE
===========================

Your quizzes will now automatically save to the dashboard!

STEP 1: Run the SQL
-------------------
Copy and paste the contents of database_setup.sql into your database:

CREATE TABLE IF NOT EXISTS quizzes (
    id INT AUTO_INCREMENT PRIMARY KEY,
    user_id INT NOT NULL,
    quiz_name VARCHAR(255) DEFAULT 'Untitled Quiz',
    quiz_data TEXT,
    total_questions INT DEFAULT 0,
    score INT DEFAULT NULL,
    status ENUM('in_progress', 'completed') DEFAULT 'in_progress',
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id) ON DELETE CASCADE
);


STEP 2: Update quiz-script.js
------------------------------
Replace the showResult() function in quiz-script.js with the one from quiz-script-patch.js

OR manually add this line before the localStorage.removeItem calls in showResult():

  // Save score to database
  if (typeof updateQuizScore === 'function') {
    updateQuizScore(score, currentQuiz.length);
  }


HOW IT WORKS
------------
1. When you generate a quiz and it loads, it automatically saves to the database
2. When you complete the quiz, the score is automatically updated
3. View all your quizzes in the dashboard
4. Delete quizzes you don't want anymore


TESTING
-------
1. Login to your account
2. Generate a quiz using drag & drop
3. Complete the quiz
4. Go to dashboard.html
5. You should see your quiz with the score!


NOTES
-----
- Quizzes are only saved if you're logged in
- If not logged in, quizzes work normally but aren't saved
- Each user only sees their own quizzes
- Dashboard shows total quizzes, completed count, and average score
